Overview

This is more of a monkey patch for Pry.

The intent of this extension is provide a way of expressing binding.pry if some_condition without Rubocop complaining or stripping the pry out leaving a bare if statement

def pry_if(flag = nil, object = nil, options = {})
  flag = yield if flag.nil? && block_given?
  return unless flag
  require 'pry' unless defined? Pry
  Pry.config.output = STDOUT
  if object.nil? || Hash === object
    Pry.start(self, object || {})
  else
    Pry.start(object, options)
  end
end
